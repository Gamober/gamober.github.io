# Fitness Diet Calculator

A bilingual (ZH/EN) fitness diet calculator based on the HaoRen SongSong Excel suite. Auto-calculates macros and meal plans for cutting and bulking diets.

## Usage

Visit [gamober.github.io](https://gamober.github.io/) to use the calculator directly.

## Features

- 15 diet plans (8 fat loss + 7 muscle gain)
- Food nutrition database (80+ items)
- FAQ section covering common diet questions
- Mobile-friendly responsive design
- Chinese / English bilingual

## Tech

Pure static HTML/CSS/JS — no frameworks, no dependencies.
